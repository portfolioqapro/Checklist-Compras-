# 🛒 Checklist de Compras

Projeto pessoal simples de um **checklist interativo de compras**, desenvolvido em HTML, CSS e JavaScript puro (sem frameworks).

## 🎯 Objetivo

Criar uma lista de compras:
- Fácil de usar 📱
- Compatível com celular e desktop 🌐
- Compartilhável via link (GitHub Pages) 🔗
- Com marcação de itens concluídos ✔️

---

## 🚀 Funcionalidades

- ✅ Marcar itens como concluídos
- 🔁 Desmarcar itens (toggle)
- 📱 Interface responsiva (mobile-friendly)
- 🌐 Acesso via navegador (sem instalação)
- 🔗 Link público compartilhável

---

## 🧱 Estrutura do Projeto

O projeto é composto por um único arquivo:
---

### 📄 HTML
Responsável pela estrutura da lista:
- Itens de compra
- Checkboxes interativos
- Organização visual simples

### 🎨 CSS
Estilização básica:
- Layout limpo
- Itens em formato de lista
- Feedback visual ao concluir (texto riscado + opacidade)

### ⚙️ JavaScript
Lógica de interação:
- Detecta quando um item é marcado
- Aplica/remover estilo de concluído (`done`)
- Funciona de forma leve e instantânea

---

## 🧪 Como usar

1. Abra o link do projeto no navegador
2. Clique nos checkboxes para marcar itens
3. Os itens marcados ficam riscados automaticamente

---

## 🌍 Publicação

Este projeto está publicado via **GitHub Pages**, permitindo acesso público através de um link.

> Exemplo: https://portfolioqapro.github.io/Checklist-Compras-/

## ⚠️ Limitações

Por ser uma versão simples:
- ❌ Não salva progresso (sem banco de dados)
- ❌ Não envia notificações
- ❌ Não possui backend
